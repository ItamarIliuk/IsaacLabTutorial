# IsaacLabTutorial

A practical workshop repository for learning **NVIDIA Isaac Lab** by building, experimenting, and training robotics simulations step by step.

This project accompanies the tutorial documentation and workshop material, helping participants move from environment setup to creating and training a quadruped locomotion task in Isaac Lab.

## What this repository covers

This repository is organized to support a complete hands-on learning path around Isaac Lab and robot learning workflows.

### Highlights

- Learn the **fundamentals of Isaac Lab** through guided examples
- Understand the **structure of simulation environments** and task definitions
- Build a **quadruped robotics environment** progressively
- Train a locomotion policy using **Reinforcement Learning**
- Explore the material interactively through **Jupyter notebooks**
- Reproduce the workshop setup using helper **shell scripts** and utilities

## Workshop methodology

The workshop follows a **learn-by-doing** methodology.

Instead of focusing only on theory, the content is structured so participants can:

1. **Set up the environment** needed to run Isaac Lab
2. **Follow guided notebooks** to understand each concept incrementally
3. **Inspect and modify Python code** to connect ideas with implementation
4. **Train and evaluate** a robotics policy in simulation
5. **Revisit the steps later** for self-paced reinforcement of the material

This approach makes the workshop suitable both for live delivery and for independent study afterward.

## Repository structure

The repository combines different kinds of material to support the workshop:

- **Python code** for examples, environment definitions, and training-related logic
- **Jupyter notebooks** for guided interactive exploration
- **Shell scripts** for setup and execution support

## Who this is for

This repository is intended for:

- Workshop participants
- Developers getting started with Isaac Lab
- Students and practitioners interested in robotics simulation and RL
- Anyone who prefers a practical, example-driven introduction

## Suggested way to use this repository

- Start with the accompanying documentation and workshop instructions
- Follow the notebooks and examples in sequence
- Run the code, inspect the outputs, and experiment with small changes
- Use the setup scripts to reproduce the intended environment
- Return to the materials after the workshop to deepen understanding

## Learning outcomes

By working through this repository, participants should be able to:

- Understand the purpose and workflow of Isaac Lab
- Navigate a robotics tutorial project structure
- Build and configure a simulated task environment
- Run and experiment with training workflows
- Connect theoretical RL concepts with executable robotics examples

## Notes

The repository is designed as a tutorial companion, so different files and branches may represent different stages of the learning journey.

If you are using this material in a workshop setting, follow the sequence recommended by the instructor or workshop guide.
